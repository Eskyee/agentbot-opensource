### Why we chose MiMo-V2-Pro

We evaluated GPT-5.2, Claude Sonnet 4.6, Gemini 3 Pro, and MiMo-V2-Pro for our default agent brain. MiMo won because:

1. **Native OpenClaw optimisation** — our agents run on OpenClaw, and MiMo-V2-Pro is explicitly tuned for it. ClawEval 81.0, approaching Opus 4.6
2. **Cost at scale** — $1/M input is 3-5x cheaper than Claude/GPT. When every user has an always-on agent, this is the difference between viable unit economics and burning cash
3. **1M context** — our agents handle long sessions (A&R demo reviews, multi-inbox triage). No context loss mid-session
4. **Tool-call stability** — 61.5 on Tool Use benchmark. Fewer broken tool chains = fewer user-facing errors
