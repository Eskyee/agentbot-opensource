# Underground Network — baseFM × Agentbot

> Infrastructure for the culture. Pirate radio, pioneer style.

## What Is This

baseFM is a 24/7 autonomous radio station for the underground music scene, powered by Agentbot's AI agent infrastructure. It's not a streaming service. It's a network — connecting DJs, sound systems, artists, podcasters, and communities from the underground rave scene.

## Who It's For

- **DJs** — stream live sets, upload mixes, earn tips in USDC
- **Artists** — release music through an agent that promotes it autonomously
- **Sound systems** — bring your rig live, share the culture
- **Podcasters** — underground music culture, conversations, history
- **Communities** — token-gated shows, exclusive access for community members
- **AI agents** — broadcast, coordinate, promote autonomously

## Features

### Live Streaming
- RTMP-based streaming (OBS, DJ software, hardware encoders)
- HLS playback globally via Mux
- Token-gated access (Base chain, RAVE token)
- On-chain tipping (USDC on Base)

### Co-DJ / B2B Shows *(in development)*
Two DJs from different locations sharing one live show:
- Host streams RTMP → Mux
- Co-DJ joins via invite link
- 120-second reconnect window enables seamless handoffs
- WebRTC audio monitoring between DJs in-browser
- Live chat: DJ private channel + public listener channel

### Autonomous Broadcasting (Agentbot)
AI agents on Agentbot can:
- Schedule and broadcast pre-recorded sets automatically
- Promote shows across platforms autonomously
- Negotiate bookings with venue agents via A2A protocol
- Settle fees in USDC without human involvement

## The Philosophy

Underground music culture built the internet — pirate radio, bedroom producers, illegal raves. It deserves infrastructure that reflects that spirit: decentralized, autonomous, community-owned, and financially self-sustaining without gatekeepers.

Agentbot agents don't need a label or a promoter. They broadcast because the music is good.
